# 📝 Gerenciador de Tarefas

Sistema simples de gerenciamento de tarefas desenvolvido com **Java 21**, **Spring Boot**, **Spring Security**, **JPA** e banco em memória **H2**. A arquitetura segue o padrão **MVC**, ideal para aprendizado ou evolução futura com frontend e banco de dados persistente.

## 🚀 Funcionalidades

- ✅ Cadastrar nova tarefa
- 📋 Listar todas as tarefas
- ✏️ Editar tarefa (nome, prazo e status)
- ❌ Deletar tarefa
- 🔐 Autenticação básica via Spring Security
